# macostimes · macOS 多时区桌面时钟

一个常驻 macOS 桌面的多时区时钟。透明无边框窗口悬浮在桌面顶部,一眼看清全球多个城市的时间。

## 特性

- 🕐 **多时区同时显示** —— 每个时钟 = 模拟表盘 + 城市名 + 数字时间
- 🌍 **时区可增删** —— 右键菜单从 14 个常用城市添加/删除
- 📏 **两种模式** —— 完整模式(表盘+数字)/ 简版模式(仅数字+城市),右键一键切换
- 🪟 **透明悬浮** —— 无边框、半透明深色,不挡视线
- ✋ **可拖动** —— 按住窗口拖到任意位置,支持跨主屏/副屏
- 📌 **可调透明度** —— 100% / 80% / 60% / 45% / 30%
- 🔄 **常驻置顶开关** —— 可关闭以降到普通窗口层
- 💾 **配置持久化** —— 时区列表、显示模式重启后保留
- ⏱️ **夏令时自动处理** —— Foundation 内置 IANA 时区库

## 环境要求

| 项目 | 要求 |
|------|------|
| macOS | 12.0 (Monterey)+ |
| 构建 | Command Line Tools(swiftc),无需安装 Xcode |

## 快速开始

```bash
make start    # 编译 + 启动(后台运行)
```

时钟会出现在桌面顶部居中。

## 常用命令

| 命令 | 作用 |
|------|------|
| `make start` | 编译 + 启动 |
| `make stop` | 停止 |
| `make restart` | 编译 + 重启 |
| `make build` | 仅编译 |
| `make clean` | 清理编译产物和日志 |

## 右键菜单

右键点击时钟窗口:

```
删除时区      ▶  选择要删除的城市
添加时区      ▶  Beijing / Tokyo / London / ... (14 个)
窗口透明度    ▶  100% / 80% / 60% / 45% / 30%
✓ 常驻置顶       切换悬浮置顶 / 普通窗口层
✓ 简版(仅数字)    切换完整表盘 / 仅数字模式
─────────
退出
```

## 配置文件

时区列表、显示模式自动持久化到:

```
~/.config/mtimes/clocks.json      # 时区列表
~/.config/mtimes/settings.json    # 显示模式等设置
```

## 技术栈

- **SwiftUI** + **AppKit**(纯原生,零第三方依赖)
- `TimelineView` + `Canvas` 绘制表盘指针(macOS 12+)
- `Foundation.TimeZone` 处理时区(含夏令时)
- `NSWindow`(`.floating` 层、`borderless`、`isMovableByWindowBackground`)实现透明悬浮可拖动窗口

## 项目结构

```
Sources/
├── ClockApp.swift          # 入口 + AppDelegate
├── WindowManager.swift      # 窗口配置 + 右键菜单 + 拖动
├── ClockStore.swift         # 时区模型 + JSON 持久化
├── AnalogClockView.swift    # Canvas 表盘绘制
├── ClockCardView.swift      # 单个时钟卡片(完整/简版)
└── ClockRowView.swift       # 横排所有时钟 + 每秒刷新
```

## 已知限制

- **贴桌面层(.desktop)无法拖动**:macOS 不向壁纸层窗口派发鼠标事件,故默认用悬浮层(.floating)。需要"沉浸感"可右键调低透明度或关闭常驻置顶。
